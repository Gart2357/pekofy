# pekofy

語尾にぺこをつけるぺこ

## インストール

```
$ pip install git+https://github.com/gaato/pekofy
```

## コンソールで使う

```
$ python -m pekofy
```

文を打ってエンターを押すとぺこがつくぺこ。
終わりたい時は ctrl-c とか ctrl-d とか押せばエラーで止まるぺこ。

## Python で使う

```py
from pekofy import pekofy

print(pekofy('そんなのってないじゃん'))
```
